Features
--------

The following list shows features aligned with performance win and also complexity.

The goal is to come to point "Freeze".

- Grouping of CSS over </head> tag
- Grouping of JS over </body> tag
- Processing of JS and CSS to new, merged files
- Compression of CSS and JS.
- Creation of individual or common pairs of above- and below-the-fold CSS
- Merge of above- and below-the-fold CSS to common above- and below-the-fold CSS (used by all pages with same CSS)
- Freeze of the common above- and below-the-fold CSS to static common above- and below-the-fold CSS
