Configuration in EM
-------------------

6 Tabs allow to setup the extension.

If you do not need special setups for some pages of your website - then please configure the extension in extension manager (EM), there's no need to install it in a template and use TypoScript-template. 

TypoScript options - apart from the API key - are **identical** with the options in extension manager

However, we document these options all in section "Configuration".
